# MHE



Program przedstawia znajdowanie rozwiązań za pomocą różnych algorytmów heurystycznych.
Aktualnie zaimplementowane algorytmy:
- Tabu
- Wspinaczkowy i Wspinaczkowy niedeterministyczny
- Pełny przegląd (Brute Force)
- Losowe próbkowanie
